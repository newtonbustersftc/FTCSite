Tips

Connecting Robot phone to Android Studio on Windows

Connect phone via USB to the computer.

Type adb tcpip 5555 in Android Studio Terminal.

Disconnect the phone.

Connect computer to the direct WiFi network of the Robot phone. The network has robot name in its name. The passphrase and IP address of the robot phone on this network can be found in “About” section of the FTC app.

Type adb connect 192.168.49.1 in Android Studio Terminal, where 192.168.49.1 is the default IP address of the Robot phone from the previous step.

Copying Robot configuration file to a different phone

Robot configuration XML files are stored in /sdcard/FIRST directory on the Robot phone.

With the first phone connected to the computer, go to Android Studio Terminal. Type adb pull /sdcard/FIRST/yourConfigName.xml where yourConfigName is the configuration name you want to copy. This will pull the configuration file to the current directory on your computer.

Disconnect the first phone from the computer. (It it is connected wirelessly, use adb disconnect.)

With the second phone connected to the computer, type adb push yourConfigName.xml /sdcard/FIRST/yourConfigName.xml. This will push the configuration file from the current directory to the right place on the second Robot phone.

Note: You can use the same approach to modify serial numbers in the configuration file after a module was replaced.

Steps to merge in SDK changes

Make sure upstream repository is set:

    git remote -v
    origin  https://github.com/newtonbustersftc/ftc_app.git (fetch)
    origin  https://github.com/newtonbustersftc/ftc_app.git (push)
    upstream        https://github.com/ftctechnh/ftc_app.git (fetch)
    upstream        https://github.com/ftctechnh/ftc_app.git (push)

If not, set the upstream repository:

    git remote add upstream https://github.com/ftctechnh/ftc_app.git

Fetch the upstream repository changes:

    git fetch upstream

Now upstream/master is a local branch.

Fetch current repository changes:

    git fetch origin

Make sure you are in master branch:

    git checkout master

Try automatic merge:

    git merge upstream/master

If automatic merge fails (which is likely), stop the merge with

    git merge --abort
